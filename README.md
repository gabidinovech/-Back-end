# 🌿 Planty — Твой онлайн сад

Студенттік жоба: AlmaU, Экономика және ІТ интеграциясы.

## 📋 Жоба сипаттамасы (API Documentation)
Бұл жоба өсімдіктерді сатуға арналған Full-stack қосымша. Төменде қолжетімді API эндпоинттары көрсетілген:

| Метод | Эндпоинт | Сипаттама |
| :--- | :--- | :--- |
| **POST** | `/api/register` | Жаңа пайдаланушыны тіркеу (Email бірегей болуы тиіс) |
| **POST** | `/api/login` | Жүйеге кіру және сессияны бастау |
| **GET** | `/api/cart` | Пайдаланушының себетін алу (JOIN арқылы өсімдік мәліметтерімен бірге) |
| **POST** | `/api/cart` | Таңдалған тауарды себетке қосу |
| **DELETE** | `/api/cart/:id` | Себеттен тауарды ID бойынша өшіру |
| **PUT** | `/api/user/update` | Пайдаланушы мәліметтерін жаңарту |

## ⚙️ Тәуелділіктерді орнату (Requirements)
Жобаның дұрыс жұмыс істеуі үшін келесі кітапханалар қажет (package.json):
* **express**: Веб-сервер құру үшін.
* **mysql2**: MySQL дерекқорымен байланыс.
* **cors**: Фронтенд пен бэкенд арасындағы сұраныстарды рұқсат ету.
* **path**: Файл жолдарын басқару.

## 🚀 Іске қосу нұсқаулығы
1. **Дерекқор**: XAMPP арқылы `planty_db` дерекқорын құрып, `planty_mysql.sql` файлын импорттаңыз.
2. **Орнату**: Терминалда `npm install` командасын орындаңыз.
3. **Қосу**: Серверді `node server.js` бұйрығымен іске қосыңыз.
4. **Кіру**: Браузерде `http://localhost:3000/index.html` мекенжайын ашыңыз.
